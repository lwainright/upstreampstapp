# Upstream Peer Support App

Upstream is a confidential, stigma‑free wellness app designed for first responders.  
It provides quick access to grounding tools, breathing exercises, peer support resources, and optional AI‑assisted support — all without tracking, accounts, or identifying information.

This version is a lightweight, static Progressive Web App (PWA) built for GitHub Pages.  
It works fully offline once loaded and can be installed on mobile or desktop.

---

## 🚨 Purpose

First responders face unique stressors that require fast, private, judgment‑free support.  
Upstream is built to provide:

- Anonymous wellness tools  
- Grounding and breathing exercises  
- Peer Support Team (PST) contact options  
- Optional AI support for talking through tough calls  
- Resource lists for EMS, fire, law enforcement, dispatch, and corrections  
- Veteran‑aligned support pathways  

No accounts. No analytics. No tracking. No admin visibility.

---

- **Home Screen** with quick access to tools and support  
- **Breathing Tools** (box breathing, paced breathing)
